# ManagementSystemTemplate -- 管理系统模板(后端)

## 技术栈

- SpringBoot
- Mybatis & Mybatis Plus
- Spring doc & Swagger UI
- Redis
- MySQL & Oracle

## 模板实现功能

- [x] job 管理
- [x] 登录认证
- [x] 用户-角色-权限管理
- [x] 邮件发送
- [x] 短信发送
- [x] 验证码, 二维码生成
- [x] 文件管理
- [x] 文件管理多数据源支持(阿里云, 七牛云...)
- [ ] 多数据源支持(MySQL, Oracle) -- 暂定
- [x] Docker 构建
- [x] Jenkins 构建
